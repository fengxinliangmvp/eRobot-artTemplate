# 欢迎访问

------
## 是什么？

基于jquery、require和artTemplate搭建的面向对象的模板程序

### 如何使用？

1. 开启服务并将代码放于服务器上（本地用的nginx）
2. 直接访问配置好的地址（localhost:50002）
